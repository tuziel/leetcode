# 69. x 的平方根 [EASY]

<https://leetcode-cn.com/problems/sqrtx/>

实现 `int sqrt(int x)` 函数。

计算并返回 *x* 的平方根，其中 *x *是非负整数。

由于返回类型是整数，结果只保留整数的部分，小数部分将被舍去。

**示例 1:**

<pre><strong>输入:</strong> 4
<strong>输出:</strong> 2
</pre>

**示例 2:**

<pre><strong>输入:</strong> 8
<strong>输出:</strong> 2
<strong>说明:</strong> 8 的平方根是 2.82842..., 
     由于返回类型是整数，小数部分将被舍去。
</pre>
