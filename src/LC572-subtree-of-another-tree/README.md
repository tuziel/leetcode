# 572. 另一个树的子树 [EASY]

<https://leetcode-cn.com/problems/subtree-of-another-tree/>

给定两个非空二叉树 **s** 和 **t**，检验 **s** 中是否包含和 **t** 具有相同结构和节点值的子树。**s** 的一个子树包括 **s** 的一个节点和这个节点的所有子孙。**s** 也可以看做它自身的一棵子树。

**示例 1:**

给定的树 s:

<pre>     3
    / \
   4   5
  / \
 1   2
</pre>

给定的树 t：

<pre>   4 
  / \
 1   2
</pre>

返回 **true**，因为 t 与 s 的一个子树拥有相同的结构和节点值。

**示例 2:**

给定的树 s：

<pre>     3
    / \
   4   5
  / \
 1   2
    /
   0
</pre>

给定的树 t：

<pre>   4
  / \
 1   2
</pre>

返回 **false**。
