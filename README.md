## 文件结构
```
|- src/  --------------------- 
    |- problems  ------------- 每个问题对应的文件夹
        |- index.js  --------- 问题对应的解题代码
        |- test.js  ---------- 测试配置
    |- task.js  -------------- 测试脚本
|- index.js  ----------------- 测试入口
```

## 命令行
```shell
# 通过子进程执行测试防止死循环
npm run test <file>

# 直接使用 mocha 测试
mocha <file>
```
